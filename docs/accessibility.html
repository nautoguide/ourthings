Accessibility Reference

Headers

The document <head> must contain:

	<meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

Forms

Forms are landmarks so the aria-label on the form is read first. Then the first input label with focus. IE the following will read : 'Search form to do something. Enter a thing to do a thing' 


<form aria-label="Search form to do something">
    <label for="searchText">Enter a thing to do a thing</label>
    <input type=text id="searchText" value="" autofocus>

    <button aria-label="Submit the search" id="findButton">Find</button>
</form>

Adding our things submit

@do.something({},{"queueBindTarget":"#findButton","queueEvent":"click"});

accessible slider


<label for="maxAge">Child's maximum age</label>
<input type="range"
       id="maxAge"
       min="0"
       max="25"
       value="25"
       step="1"
       aria-valuemin="1"
       aria-valuemax="25"
       aria-valuenow="25"
>
<output for="maxAge" id="maxAgeValue">25</output>

@elements.innerHTML({"targetId":"#maxAgeValue","html":"{{!queue.getElement('#maxAge').value}}"},{"queueBindTarget":"#maxAge","queueEvent":"input"});
